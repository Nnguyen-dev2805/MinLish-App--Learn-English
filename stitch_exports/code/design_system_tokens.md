## Component Guidance

- Filled buttons: high-contrast teal with white text.
- Tonal buttons: mint background with dark teal text.
- Word cards: 24dp corners, subtle tonal elevation, optional audio icon.
- Progress cards: thin primary border or surface-container tonal layer.
- Linear progress: 8dp high, rounded caps, primary bar over secondary track.
- Inputs: outlined style, 12dp radius, active primary border.
- Bottom navigation: Material 3 NavigationBar, active tonal pill around icon.
